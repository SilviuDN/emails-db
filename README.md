# Email database:
- a Node.Js app with mySql

## get total users in database:
- npm start

## Environmental variables: 
    MYSQL_HOST=localhost
    MYSQL_USER --> your user
    MYSQL_PASSWORD --> your password

## Dependencies:
    "dotenv": "^10.0.0",
    "express": "^4.17.1",
    "faker": "^5.5.3",
    "mysql": "^2.18.1"
